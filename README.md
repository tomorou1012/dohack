# SafeBridge - 災害時AIガイド＆地域連携アプリ

![SafeBridge Logo](public/icon-192.png)

**災害時の迅速な避難をサポートする総合ガイドアプリ**

SafeBridgeは、災害時に住民が迅速かつ安全に避難できるよう、AIによるガイダンス、リアルタイム避難所マップ、天気警報情報などを一元的に提供するWebアプリケーションです。

## 🎯 主な機能

### ✅ 実装済み機能

## 🚀 セットアップと起動

### 必要な環境
- Node.js 18以上
- npm または yarn

### インストールと起動

```bash
# リポジトリをクローン
git clone <repository-url>
cd safebridge-disaster-app

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスしてください。

### その他のコマンド

```bash
# プロダクション用ビルド
npm run build

# 型チェック
npm run typecheck

# プロダクション版を起動
npm run start
```

## ⚠️ 重要な注意事項

**このアプリは災害時の補助ツールです。生命に関わる緊急事態では、必ず以下の番号に直接連絡してください：**

- **消防・救急**: 119
- **警察**: 110
- **災害用伝言ダイヤル**: 171

SafeBridgeは情報提供を目的としており、公式な災害情報や避難指示は、自治体や気象庁の公式発表を必ず確認してください。
